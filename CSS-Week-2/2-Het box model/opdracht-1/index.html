Block boxes


will break onto a new line;

will become as wide as its container;

filling up 100% of the space available;

width and height properties are respected

will break onto a new line.

Padding, margin and border will cause other elements to be pushed away from the box



Inline boxes


will not break onto a new line;

width and height properties are not respected

will break onto a new line.

vertical Padding, margin and border will apply but will not cause other elements to be pushed away from the box

horizontal Padding, margin and border will apply and will cause other elements to be pushed away from the box
