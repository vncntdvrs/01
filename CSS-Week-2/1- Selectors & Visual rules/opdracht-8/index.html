Cascade

Cascade implies that the order of CSS rules matter;
when two rules apply that have equal specificity the one
that comes last in the CSS is the one that will be used.

In this example, we have two rules that could apply to the h1.
The h1 ends up being colored blue even though the first h1 selector is set to red



Inheritance

Inheritance means that some CSS properties by default
inherit values set on the current element's parent element,
and some don't. This can also cause some behavior that
you might not expect.

Some CSS property values set on parent elements are
inherited by their child elements, and some aren't.
For example, if you set a color and font-family on an element,
every element inside it will also be styled with that color
and font, you've applied different color and font values directly
to them. by using for instance a span element
