Type selectors

A Type Selector (sometimes referred to as an Element Type Selector) matches elements with the corresponding
element node name, such as p, span and div Type selectors are generally used to make “broad stroke”
changes to the style of a site.

Descendant selectors

The descendant selector matches all elements that are descendants of a specified element.

Descendant selectors allow you to limit the targeted elements to the ones who are descendants
of another element.

class selector

The class selector selects HTML elements with a specific class attribute.

HTML elements can also refer to more than one class
